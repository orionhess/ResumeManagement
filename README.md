## Description
We plan to make a resume management tool for mixing and matching different aspects of your resume. Perhaps you want all of your projects and language skills for a CS-related application, but don’t need those and would prefer to highlight soft skills and other work experience for a different application.

Our program will store and allow editing of these different pieces of your essay, then picking certain aspects of them and compiling them into a pdf, likely using latex. 

For CI/CD, we will be using pytest for specific module tests and integration testing of the program as a whole. Github actions on Ubuntu-Latest will be used for automated running of those tests. For checking and formatting, we will use something similar to pre-commit hooks with Ruff.
